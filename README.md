## Sobre o Projeto

Projeto utiliza algumas das seguintes tecnologias:

Spring Tools
Lombok
MySQL
Angular 9
Tema do Creative
Bootstrap 4
Docker
Swagger

## Como rodar o projeto na sua máquina

1. Instale o NodeJs de [NodeJs Official Page](https://nodejs.org/en).
2. Abra o terminal
3. Vá para o diretório do projeto
4. Execute o comando: ```npm install -g @angular/cli```
5. Então execute: ```npm install```
6. E depois: ```ng serve```
7. Pronto, basta navegar para o link: [http://localhost:4200/](http://localhost:4200/)
